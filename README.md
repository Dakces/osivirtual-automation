# OsiVirtual Automation

Framework de automatización con Selenium + Python para el sistema OsiVirtual.

- Ingreso a Ventanilla Virtual (VVO)
- Manejo de nueva pestaña
- Resolución de modal inicial

## Tecnologías
- Python
- Selenium
- Pytest
- Page Object + Flows
- dotenv
